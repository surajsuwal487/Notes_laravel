#PHP and Laravel Standard Namings
   Model
      Singular with first letter capital.
      Example: Book, BookReview.
   
   Controller
      Singular is prefered.
      Example: BookController
      Move controller code to Model, Service, Action and Job.
   
   Factory
      Singular is prefered.
      Example: BookFactory.
   
   Seeder
      Singular is prefered.
      Example: BookSeeder.
   
   Migration
      create_book_reviews_table

   Function
      public function report.
      public function getFinancialReportData.
      Use return type in function.
